# Aperçus pour le README

Le dépôt peut inclure ici des **captures d’écran** référencées par le README racine.

1. Après un run local, copier par exemple depuis `reports/figures/` :
   - `preview_forecast.png` ← typiquement un `03_test_forecast_*`
   - `preview_metrics.png` ← typiquement un `06_tableau_metriques_*`
2. Les coller dans ce dossier (`docs/assets/`).
3. Les images sont **optionnelles** : sans elles, le README reste valide (lien « ex. » ou section masquée).

Taille conseillée : largeur ~900–1200 px pour une lecture confortable sur GitHub.
